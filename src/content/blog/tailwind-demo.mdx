## 代码说明

要在 Astro 中使用 Tailwind CSS，你需要：

1. 安装依赖：`npm install tailwindcss @tailwindcss/vite`
2. 在 `astro.config.mjs` 中添加 Vite 插件
3. 在全局 CSS 文件中导入 `@import "tailwindcss"`

完成后，你就可以在 `.astro`、`.mdx` 文件中使用 Tailwind 的所有工具类了！
